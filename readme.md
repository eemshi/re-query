# re-query

A Github repo querying app, built with React / Next / Express

__Requirements__
- Node v10.15.0 or higher
- [Redis](https://redis.io/topics/quickstart)

__Install dependencies__

`npm install`

__Start Redis__

`redis-server`

__For dev-ing with live server reload:__

`npm run dev`

__Otherwise,__

`npm start`
